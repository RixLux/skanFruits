    v1.1.0: UI Refinement & Thresholding

    • Rombak Alur Navigasi:
      • Pembuatan ResultScreen sebagai halaman penuh untuk menampilkan detail analisis.
      • Implementasi tombol "Capture" manual pada ScannerScreen.
      • Navigasi otomatis ke ResultScreen setelah pengambilan gambar (Kamera/Galeri).
    • Logika Threshold 80%:
      • Penambahan logika kondisional pada ScannerController atau FruitAnalyzerService: if (confidence < 0.80).
      • Implementasi UI peringatan khusus pada ResultScreen untuk hasil dengan keyakinan rendah.
    • Feedback Instan:
      • Implementasi modal loading sederhana saat proses analisis berlangsung sebelum transisi halaman.
